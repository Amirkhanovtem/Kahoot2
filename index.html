Полное руководство по Kahoot

Вход в систему и создание викторин

Уважаемые коллеги, Kahoot! предназначен только для использования в Учебном центре. Использование в личных целях запрещено. Поскольку аккаунт является общим, просьба не удалять и не корректировать чужие опросы.

Часть 1: Вход в Kahoot

Перейдите на сайт kahoot.com и нажмите Log In.

Ввод логина и пароля

Введите логин и пароль. При повторном входе с того же устройства авторизация произойдет автоматически.

Получение доступа с нового устройства

Если вы используете новое устройство, нажмите Click here to get access.

Ввод корпоративной почты

Введите корпоративную почту: yuliya.grausbard@airastana.kz и нажмите I consent.

Важно: Использование личной почты запрещено!

Завершение входа

Нажмите Continue и вы будете перенаправлены на главную страницу Kahoot.

Часть 2: Создание и запуск викторины

1
Создание нового квиза

Справа виден список всех ранее созданных опросников, нужно нажать "Create".

2
Выбор типа игры

Выберите из доступных вариантов создания игры. Рекомендуется выбрать Kahoot для создания интерактивных презентаций с опросами и викторинами.

3
Название и добавление вопросов

В верхней части введите название викторины, слева добавляйте вопросы нажав "Add question".

4
Заполнение деталей вопроса

При добавлении вопроса выберите тип, затем заполните все нужные поля: вопрос, варианты ответа, отметьте верный ответ. При необходимости можно добавить изображение.

Типы вопросов: Quiz (викторина), True or false (верно/неверно), Type answer (ввод ответа), Poll (опрос)
5
Экран с добавленными вопросами

Просмотрите экран со всеми вопросами, которые вы внесли. Здесь можно редактировать, удалять или добавлять новые вопросы.

6
Добавление названия и описания

Предоставьте четкое название и описание для данной игры. Это поможет сделать ваш Kahoot более видимым и понятным для других пользователей.

7
Сохранение викторины

После внесения всех вопросов нажмите "Save" для сохранения опросника (в верхнем правом углу). Ваша викторина готова к запуску!

Наиболее подходящий вариант для проведения: Present
8
Выбор режима презентации

Выберите подходящий режим для проведения вашей викторины. Рекомендуется режим "Host live" для проведения викторины в реальном времени.

Режимы:
  • Preview - предварительный просмотр
  • Host live - проведение в реальном времени
  • Presentation - режим презентации
  • Share - поделиться с другими
9
Экран с пин-кодом игры

Запускается окно с пин-кодом на игру (каждый раз новый). Начиная с этого момента можно показывать этот экран участникам для подключения.

Как подключиться участникам:
  • Зайти на сайт kahoot.it
  • Ввести пин-код игры
  • Ввести свое имя
  • Дождаться начала игры

По вопросам обращайтесь:

Complete Kahoot Guide

Login and Quiz Creation

Dear colleagues, Kahoot! is intended for use only within the Training Center. Personal use is strictly prohibited. Since the account is shared, please do not delete or modify other people's surveys.

Part 1: Kahoot Login

Go to kahoot.com and click Log In.

Enter Username and Password

Enter your username and password. When logging in again from the same device, authorization will occur automatically.

Access from New Device

If you are using a new device, click Click here to get access.

Enter Corporate Email

Enter your corporate email: yuliya.grausbard@airastana.kz and click I consent.

Important: Personal email addresses are prohibited!

Complete Login

Click Continue and you will be redirected to the Kahoot home page.

Part 2: Creating and Launching Quiz

1
Create New Quiz

On the right, you can see a list of all previously created surveys, click "Create".

2
Choose Game Type

Choose from available game creation options. It's recommended to select Kahoot for creating interactive presentations with polls and quizzes.

3
Title and Adding Questions

Enter the quiz title at the top, add questions on the left by clicking "Add question".

4
Fill Question Details

When adding a question, select the type, then fill in all required fields: question, answer options, mark the correct answer. You can add an image if needed.

Question types: Quiz, True or false, Type answer, Poll
5
Screen with Added Questions

Review the screen with all questions you've entered. Here you can edit, delete, or add new questions.

6
Add Title and Description

Provide a clear title and description for this game. This will help make your Kahoot more visible and understandable to other users.

7
Save Quiz

After adding all questions, click "Save" to save the survey (in the upper right corner). Your quiz is ready to launch!

Most suitable option for conducting: Present
8
Choose Presentation Mode

Choose the appropriate mode for conducting your quiz. The "Host live" mode is recommended for real-time quiz sessions.

Modes:
  • Preview - preview mode
  • Host live - conduct in real-time
  • Presentation - presentation mode
  • Share - share with others
9
Game Pin Code Screen

A window with a game pin code will launch (new each time). From this moment, you can show this screen to participants for connection.

How participants connect:
  • Go to kahoot.it
  • Enter the game pin code
  • Enter their name
  • Wait for the game to start

2-бөлім: Викторина жасау және іске қосу

1
Жаңа викторина жасау

Оң жақта барлық бұрын жасалған сауалнамалардың тізімі көрінеді, "Create" батырмасын басу керек.

2
Ойын түрін таңдау

Қолжетімді ойын жасау нұсқаларынан таңдаңыз. Сауалнамалар мен викториналары бар интерактивті презентациялар жасау үшін Kahoot таңдау ұсынылады.

3
Атауы және сұрақтар қосу

Жоғарғы бөлікте викторина атауын енгізіңіз, сол жақта "Add question" басып сұрақтар қосыңыз.

4
Сұрақ мәліметтерін толтыру

Сұрақ қосқан кезде түрін таңдаңыз, содан кейін барлық қажетті өрістерді толтырыңыз: сұрақ, жауап нұсқалары, дұрыс жауапты белгілеңіз. Қажет болса сурет қоса аласыз.

Сұрақ түрлері: Quiz (викторина), True or false (дұрыс/бұрыс), Type answer (жауап енгізу), Poll (сауалнама)
5
Қосылған сұрақтармен экран

Сіз енгізген барлық сұрақтармен экранды қарап шығыңыз. Мұнда сұрақтарды өңдеуге, жоюға немесе жаңаларын қосуға болады.

6
Атау мен сипаттама қосу

Бұл ойын үшін нақты атау мен сипаттама беріңіз. Бұл сіздің Kahoot-ты басқа пайдаланушылар үшін көрінетін және түсінікті етуге көмектеседі.

7
Викторинаны сақтау

Барлық сұрақтарды қосқаннан кейін сауалнаманы сақтау үшін "Save" басыңыз (жоғарғы оң бұрышта). Сіздің викториналыңыз іске қосуға дайын!

Өткізу үшін ең қолайлы нұсқа: Present
8
Презентация режимін таңдау

Викториналыңызды өткізу үшін қолайлы режимді таңдаңыз. Нақты уақытта викторина сессиялары үшін "Host live" режимі ұсынылады.

Режимдер:
  • Preview - алдын ала қарау режимі
  • Host live - нақты уақытта өткізу
  • Presentation - презентация режимі
  • Share - басқалармен бөлісу
9
Ойын пин-коды экраны

Ойын пин-коды бар терезе ашылады (әр уақытта жаңа). Осы сәттен бастап бұл экранды қатысушыларға байланыс үшін көрсете аласыз.

Қатысушылар қалай қосылады:
